Dropbox

Dropbox is file sharing done right. They offer 2GB of free storage to start, and you can get bonus storage up to something like 16GB. You download the app and drag and drop into your dropbox folder. It then syncs to the cloud and can be downloaded from any other machine with the app or through their web interface. You can also have multiple people upload to one account to share content.

Internet Explorer

This is on this list simply as a reference point. The only reason someone should use Internet Explorer (IE) is if they have no other option. IE is consistently the last browser to adopt new standards, typically comes in last in speed tests, doesn't have any virus protection advantage, and has no customization options. Using IE is a sign that you're new to the internet, so save yourself the embarassment and get a new browser.
